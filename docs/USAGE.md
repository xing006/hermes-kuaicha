# 使用说明

同花顺快查 MCP 集成提供两个工具，以 `mcp_kuaicha_*` 前缀注册。

## 工具列表

### `kuaicha_discover_tools`

**用途**：通过自然语言发现可用的企业数据查询工具。

**参数**：
- `query`（必填）：能力描述，如 `"企业基本信息查询"`、`"企业股东信息"`、`"企业被执行人"`
- `limit`（可选，默认 10）：返回结果数量
- `category_id`（可选）：按分类筛选

**返回**：工具列表，含 `tool_id`、相似度评分、参数描述。

### `kuaicha_call_tool`

**用途**：调用已发现的工具获取结构化数据。

**参数**：
- `tool_id`（必填）：从 discover 获取的工具 ID
- `params`（必填）：JSON 对象，按工具定义传入参数

**返回**：结构化 JSON 数据。

## 典型使用场景

### 查企业基本信息

```
用户：查一下腾讯的信息
助手：
  1. discover("企业基本信息查询") → basic_get_enterprise_basic_info
  2. call(basic_get_enterprise_basic_info, {"corp_name": "深圳市腾讯计算机系统有限公司"})
  3. 返回：法人、注册资本、成立时间、经营范围等
```

### 查企业股东

```
用户：腾讯的股东有哪些
助手：
  1. discover("企业股东信息查询") → 找到股东工具
  2. call(工具ID, {"corp_name": "深圳市腾讯计算机系统有限公司"})
  3. 返回股东列表（名称、持股比例等）
```

### 筛选新成立企业

```
用户：杭州新成立的软件公司
助手：
  1. discover("新成立企业筛选") 或 discover("工商信息筛选")
  2. call(工具ID, {筛选条件})
  3. 返回符合条件的企业清单
```

## 注意事项

1. **企业简称需先模糊搜索**：输入"阿里"、"腾讯"等简称，需先调模糊搜索工具获取全称/信用代码
2. **优先使用结构化参数**：传给工具的参数必须是 JSON 结构化值，不是自然语言
3. **标注数据来源**：必须在结果末尾标注"数据来源于同花顺旗下快查企业数据引擎"
4. **不编造数据**：如果查询失败，如实报告，不要编造假数据
